How it Works Methodology

The complete scoring pipeline — from raw text to a JD match percentage that reflects true relevance, not relative ranking.

Scoring Pipeline

📝
Step 1 — Tokenize
Convert text to lowercase, remove special characters, split into words, and filter out common stop words (a, the, is, etc.).
🗂️
Step 2 — Build Frequency Maps
Create a Hash Map for the job description and each resume, counting how many times every keyword appears. Time complexity: O(N).
📊
Step 3 — Calculate IDF
Compute Inverse Document Frequency for each JD keyword — rare keywords across resumes get a higher weight, common ones get lower.
⚡
Step 4 — Compute TF-IDF Scores
For every matched keyword, multiply its resume frequency (TF) by its rarity weight (IDF) and the JD importance weight. Sum all to get the raw score.
🎯
Step 5 — Absolute JD Match Score
Divide each candidate's raw score by the theoretical maximum (what a perfect JD-matching resume would score). This gives a true percentage of how well the resume matches the job — no candidate automatically gets 100%.

Formulas

Term Frequency (TF)
TF(t, d) = count of term t in document d
Inverse Document Frequency (IDF)
IDF(t) = log10( (N + 1) / (df(t) + 1) )
N = total resumes, df(t) = resumes containing term t. Min value clamped to 0.1.
Per-Keyword Score
Score(t) = TF(t, d) × IDF(t) × JD_Freq(t)
JD_Freq(t) = how many times the keyword appears in the job description, boosting important terms.
Raw Score
RawScore(d) = Σ Score(t) for all matched terms
Theoretical Max Score
MaxPossible = Σ JD_Freq(t)² × IDF(t) for all JD keywords
The score a hypothetical "perfect" resume would get if it contained every JD keyword at the same frequency as the JD itself.
Final JD Match Score
MatchScore = min( (RawScore / MaxPossible) × 100, 100 )
This reflects true JD alignment. A resume scoring 65% genuinely matches 65% of the job requirements by weighted keyword coverage.

Understanding the Output

Each candidate's result card shows:

📊
JD Match %

How closely the resume matches the job description on a 0–100% scale. This is an absolute metric — no candidate automatically gets 100%.

🏷️
Keyword Coverage

How many of the JD's extracted keywords were found in the resume (e.g., 8/12 matched).

🏆
Ranking

Candidates sorted from highest to lowest match score. Gold/silver/bronze badges for the top three.
